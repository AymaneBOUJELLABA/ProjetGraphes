Introduction Interface Creer un graphe Algorithmes

Cette application nous permettera de créer des différent types de graphes (orienté, pondéré,...) tous on appliquant différent algorithmes sur ces graphes

L'interface ce compose de 3 parties : le premier permet de lancer les algorithmes sur le graphe dèja créer, la deuxième où l'utilisateur peut dessiner le graphe et déplacer les noeuds et le dernier affiche le résultat
